# FAQ

## Is Lazarus a web app?

The main user experience is Telegram. The web admin panel is for internal operators.

## Is the source code public?

No. The source code is private. This docs repository is public so users can understand what Lazarus is building.

## Does Lazarus custody funds?

Lazarus can generate or store imported wallet keys for users who choose that flow. Private keys are encrypted before storage. Users can also add watch-only wallets for tracking without signing.

## Is live trading available?

Live trading is implemented as a controlled capability but should be rolled out carefully. Early access should remain allowlisted and tiny-funded until operational validation is complete.

## What is a CTO token?

CTO means community takeover. In this context, it usually refers to a token whose original team appears inactive or gone, while a community or new participants attempt to revive interest.

## Is a high score a buy signal?

No. A high score means Lazarus detected stronger conditions worth reviewing. Users still need to verify contract risk, liquidity, taxes, ownership, chart behavior, and market context.

## What chains are supported first?

Base and Ethereum mainnet. Base is the priority launch chain because transaction costs are lower.

## What does Premium include?

The planned Premium plan includes up to five wallets and costs 0.05 ETH per month.

## How do referrals work?

Users can share a referral code. The planned referral program pays 25% of attributed subscription revenue and 25% of the platform transaction fee share from referred user volume.
