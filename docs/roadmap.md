# Roadmap

This roadmap is public-facing and intentionally avoids private implementation details.

## Phase 1: Foundation

Status: in progress / mostly wired.

- Multi-worker backend.
- Telegram user bot.
- Public signal notifier.
- Multi-tenant database model.
- Encrypted user wallets.
- Billing foundation.
- Referral ledger.
- Admin dashboard.
- Paper-mode trading.
- Portfolio/PnL snapshots.

## Phase 2: Production Hardening

- Complete VPS deployment hardening.
- Confirm SSL and domain routing for app/admin surfaces.
- Verify migrations on staging and production.
- Add backup/restore process.
- Add operational dashboards.
- Improve alerting around worker health.
- Add live preflight checks.

## Phase 3: Public Signal Launch

- Launch public Telegram signal channel.
- Tune scoring thresholds.
- Add cooldown rules.
- Review post formatting on mobile.
- Collect feedback from early users.

## Phase 4: Controlled User Bot Launch

- Invite first users.
- Keep trading in paper mode by default.
- Validate onboarding, wallets, billing, referrals, and support flows.
- Tune guided presets.

## Phase 5: Live Trading Beta

- Operator-only testing.
- Fork validation.
- Tiny-funded live entries and exits.
- Strict allowlist.
- Clear risk acknowledgement.
- Daily reconciliation.

## Phase 6: Broader Premium Launch

- Premium subscription rollout.
- Referral payouts.
- Public docs expansion.
- More analytics.
- More supported chains/DEXes only after Base/Ethereum are stable.
